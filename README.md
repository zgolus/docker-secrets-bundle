Docker Secrets Bundle
=======================

#### Installation:

Add repository to `composer.json`:

```
{
    ...
    "repositories": [
        ...
        {
            "type": "vcs",
            "url": "git@github.com:zgolus/docker-secrets-bundle.git"
        }
    ]
    ...
}
```
Install the bundle:
```
composer require 'zgolus/docker-secrets-bundle:^1.0'
```
